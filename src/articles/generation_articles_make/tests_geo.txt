RESOUMETTRE LE SITEMAP (après modification)
Si le sitemap a été modifié (ex : ajout des balises image),
il faut le resoumettre manuellement pour accélérer l'indexation.
Sans resoumission, Google recrawle seul mais ça peut prendre plusieurs semaines.

  1. Aller sur https://search.google.com/search-console/
  2. Menu gauche → Sitemaps
  3. Si https://yanalabs.fr/sitemap.xml est déjà listé
     → cliquer dessus → Resoumettre
  4. Si pas encore listé
     → coller l'URL dans le champ → valider
  → Le statut passe à "Succès" en quelques minutes
  → L'indexation des images prend quelques jours
